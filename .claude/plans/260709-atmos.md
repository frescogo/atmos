# Port placar.pde to pico-atmos

Target: `atmos/main.atm` (new dir in this repo), run with
`cd atmos && atmos main.atm` (relative asset paths:
`../data/...`).
Reference idioms: pico-rocks `main.atm`
(https://github.com/atmos-lang/pico-rocks):
`require "atmos.env.pico"`, `loop on :draw`,
`watching`, `:key.dn [key=...]`, `[ '%', x=, y= ]`.

Processing grid `W=width/11`, `H=height/8` maps to `%` coords:
`x = col/11`, `y = row/8`.
Window: `pico.set.dim ['!', w=1000, h=600]`.

## Environment (verified 2026-07-09)

- `atmos` at `/usr/local/bin/atmos`; `atmos.env.pico` OK at
